# General

###### Docs: https://symfony.com/doc/4.1/

###### Directories:
config - config
src - put code here
templates - twig templates

bin - binaries/executables
var - cache and logs
vendor - third party code
public - publicly accessible files

###### Running a server:
```
run server: php bin/console server:run
```
view page: http://127.0.0.1:8000/
